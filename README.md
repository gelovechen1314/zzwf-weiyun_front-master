# 售电云运维管理系统

> A Vue.js project

## 安装

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run
npm run dev
```

## 依赖项目

* [bootstrap-vue](https://bootstrap-vue.js.org)
* [vuex](https://vuex.vuejs.org)
* [axios](https://github.com/axios/axios)
